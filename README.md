# ❌⭕ Jogo da Velha – Java (Console)

Aplicação desenvolvida em Java com foco em lógica de programação, manipulação de arrays bidimensionais e controle de fluxo.

O projeto implementa o clássico Jogo da Velha em ambiente de console, permitindo partidas entre dois jogadores no mesmo terminal.

---

## 🎯 Objetivo

- Praticar lógica de programação
- Trabalhar com matriz `char[][]`
- Aplicar controle de fluxo com loops e condicionais
- Implementar validação de entrada
- Tratar exceções (`NumberFormatException` e `IndexOutOfBoundsException`)

---

## 🧠 Conceitos Aplicados

- Programação estruturada
- Métodos estáticos
- Laço `while`
- Estrutura `do-while`
- Tratamento de exceções
- Manipulação de matriz 3x3
- Controle de estado do jogo
- Verificação de vitória por:
  - Linhas
  - Colunas
  - Diagonais
- Condição de empate

---

## 🏗 Estrutura do Projeto

- Main.java

---

O projeto é composto por um único arquivo contendo:

- Inicialização do jogo
- Preparação do tabuleiro
- Validação de jogadas
- Verificação de vencedor
- Controle de reinício de partida
